# Matemáticas I
Este curso cubre los fundamentos del cálculo y álgebra lineal necesarios para la ingeniería.

## Subapartados
- **Teoría/Problemas:** Conceptos teóricos y resolución de ejercicios.
- **Prácticas:** Uso de software matemático (ej. MATLAB).
- **AAD:** Actividades dirigidas por el profesor.

## Bibliografía
- Larson, R., Edwards, B. H. (2010). Cálculo I. McGraw-Hill.
- Strang, G. (2006). Linear Algebra and its Applications. Thomson.
